The price checking feature helps you understand the craft cost of minions before buying or selling them. This information is crucial for setting competitive auction prices and evaluating deals.

## Overview

The `/prices` command provides instant access to minion craft costs directly in Discord. The data helps you:

- Price your auctions competitively
- Evaluate if an auction is a good deal
- Understand the baseline value of minions
- Make informed purchasing decisions

## Using the Prices Command

### Basic Command

```
/prices
```

Without any parameters, this command displays all minion types and tiers with their craft costs in a paginated format.

### Filtered Search

You can filter results by specifying minion type and/or tier:

```
/prices type: Clay tier: 11
```

### Parameters

<ParamField path="type" type="string" optional>
  The type of minion to check the price for (e.g., "Clay", "Snow", "Cobblestone")
</ParamField>

<ParamField path="tier" type="integer" optional>
  The tier of the minion (1-12)
</ParamField>

## Price Display Format

Each minion entry shows:

- **Minion emoji**: Visual identification
- **Minion name**: Type and tier (e.g., "Clay Minion XI")
- **Craft cost**: Formatted price (e.g., "`1.5M`", "`250K`")

## Understanding Craft Costs

### What is Craft Cost?

Craft cost represents the total resource value needed to craft a minion from scratch, including:

- Base materials for Tier 1
- Upgrade costs for each tier
- Resource prices based on current market values

### How to Use Craft Costs

## Technical Details

### Data Source

Prices are stored in the Minion model database with the following structure:

```typescript
model Minion {
  id             String    @id
  name           String
  generator      String
  generator_tier Int
  maxTier        Int
  craftCost      Float     @default(0)
  sellers        Auction[]
}
```

The `craftCost` field is pre-calculated and stored for quick retrieval.

### Price Formatting

Prices are formatted using the `formatMinionPrice` utility function, which:

- Converts large numbers to K (thousands) or M (millions)
- Rounds to appropriate decimal places
- Adds formatting for readability
