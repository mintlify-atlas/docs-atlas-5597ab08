  <h2 className="text-2xl sm:text-3xl font-semibold text-gray-900 dark:text-white mb-4">
    Quick start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get up and running with MinionAH Bot in just a few steps
  </p>

  <Steps>
    <Step title="Add the bot to your Discord server">
      Invite MinionAH Bot to your Discord server using the invite link. The bot requires permissions to send messages and use slash commands.
      
      <Note>
        Make sure you have "Manage Server" permissions to add the bot to your Discord server.
      </Note>
    </Step>
    
    <Step title="Link your MinionAH account">
      Use the `/discord link` command to connect your Discord account with your MinionAH account. This enables personalized features like viewing your auctions and managing notifications.
      
      ```bash Discord Command
      /discord link
      ```
    </Step>
    
    <Step title="Create your first auction">
      Start creating minion auctions directly from Discord using the `/auctions create` command.
      
      ```bash Discord Command
      /auctions create
      ```
      
      <Accordion title="What happens next?">
        The bot will guide you through an interactive form where you can:
        - Select the minion type and tier
        - Set your price
        - Specify quantity
        - Add optional features (infusion, free will)
        - Mark as negotiable
      </Accordion>
    </Step>
    
    <Step title="Search for auctions">
      Browse available minion auctions with powerful filtering options.
      
      ```bash Discord Command
      /auctions search
      ```
      
      The search command supports filtering by minion type, tier, price range, and special features.
    </Step>
  </Steps>
</div>
